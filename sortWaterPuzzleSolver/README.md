# Инструкция по запуску

## 1. Проверьте версию Java

Для корректной работы программы требуется **Java версии 17 или выше**.

```bash
java --version
```

Если Java не установлена или версия ниже 17, необходимо её установить:
**[Oracle JDK](https://www.oracle.com/java/technologies/downloads/)**

## 2. Компиляция программы

Перейдите в директорию с исходным кодом (`sortWaterPuzzleSolver/src`) и выполните компиляцию:

```bash
cd src # переход в директорию с исходным кодом
javac *.java # компиляция всех Java-файлов
```
При успешной компиляции в директории появятся файлы с расширением `.class`.

## 3. Запуск программы

```bash
java SortColorPuzzle
```

При запуске программа начнёт искать решение для тестовых данных указанных в тестовом задании

```java
new Object[][]{
        {4, 4, 10, "Two"},
        {8, 12, 8, "One"},
        {9, 5, 7, 10},
        {5, "Two", 3, 5},
        {7, 8, 11, 6},
        {"Two", "One", 12, 12},
        {11, 8, 7, 4},
        {"One", 3, 11, 10},
        {9, 9, 7, 10},
        {11, 6, "Two", 6},
        {3, 9, 6, 4},
        {"One", 12, 3, 5},
        {},
        {}
}, 4
```

## 4. Тестирование и изменение тестовых данных

Для изменения входных данных отредактируйте файл `SortColorPuzzle.java`.

Входные данные должны содержать следующее:

1. **Двумерный массив `Object[][]`** размером NxV, где:
    - `N` — количество пробирок
    - `V` — ёмкость каждой пробирки
    - Элементы могут быть числами, строками или символами (при остальных типах будет возникать исключение, указывающее на неправильный тип). Типы не обязаны совпадать (это значит, что в входных данных могут быть одновременно и числа и строки)
    - Пробирки без цветов обозначаются как `{}`

2. **Целое число** — фиксированная ёмкость каждой пробирки. Должно быть положительным числом
