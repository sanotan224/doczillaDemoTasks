# Инструкция по запуску

## 1. Перейти в директорию проекта
```bash
cd fileSharingApp/backend
```

## 2. Установить зависимости
```bash
npm install
```

## 3. Запустить в режиме разработки
```bash
npm run dev
```

Или собрать и запустить
```bash
npm run build
npm start
```
